# Compliance Checklist

- [ ] Runtime model is open-weight and publicly available
- [ ] Runtime model release date is within the allowed rules window
- [ ] Every dataset in the winning path has a recorded source and license
- [ ] Notebook runs with internet disabled
- [ ] Weights, prompts, configs, and helper artifacts load cleanly
- [ ] The notebook always returns one integer in the valid range
- [ ] Timeout and crash handling are explicit
- [ ] Local or AWS rerun config for the exact submission is logged
- [ ] No private code or inaccessible artifacts are required to reproduce the result
